Tasks

Main program creates three child processes and lets the run concurrently. It waits for them to return and prints their exit statuses.

Child 1:
Calculate and display the average marks for of a class of 10 students. Marks are input via standard input (keyboard).

Child 2:
Load a program called "wc" to count the number of words in a file. file1 is passed to the program via command line argument.

Child 3:
Modify file2 by inserting the string "This is the updated version." at the beginning of the file, and then replacing all occurrences of the word "examine" with the word "check". file2 is passed to the program via the command line argument.
